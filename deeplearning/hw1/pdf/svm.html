Inline Question 1

It is possible that once in a while a dimension in the gradcheck will not match exactly. What could such a discrepancy be caused by? Is it a reason for concern? What is a simple example in one dimension where a gradient check could fail? How would change the margin affect of the frequency of this happening? Hint: the SVM loss function is not strictly speaking differentiable

$\color{blue}{\textit Your Answer:}$ Yes, The loss function of svm : $max(0, s_j - s_{y_i} + \Delta t)$ is not differentiable at 0. Then at this point there will be different gradient. Given $s_j - s_{y_i} + \Delta t = h/2$, the numerical gradient is $\frac{0 + (h/2 + h)}{2*h} = \frac{3}{4}$, but the analytic gradient is 1. The method of change the margin affect is that we set a small threshold, any value less than the threshold is treated as 0.
